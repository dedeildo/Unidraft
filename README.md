UniDraft

Um Sistema de peneiras para alunos da UFRPE com o intuito de facilitar a integração e incentivar os alunos à prática de atividades esportivas.

Sobre: O UniDraft tem como objetivo criar um sistema simples e acessível para que os discentes da instituição possam realizar atividades esportivas de seus interesses. Temos como objetivo criar um ambiente mais inclusivo e confortável para todos que desejam participar de tais práticas corporais.
